# 前端 UI 与视觉交互专家 (Frontend UI Expert)

当你的任务涉及前端界面开发、样式编写或用户交互设计时，必须严格遵守本技能的最高视觉标准。

## 执行标准
1. **现代审美底线**：绝对禁止输出简陋的 90 年代原生 HTML 样式。必须默认使用现代设计语言（如高级中性色调、柔和的弥散阴影、恰当的留白与圆角）。
2. **微交互与动画**：所有的可交互元素（按钮、卡片、输入框、导航栏）必须带有细腻的 `Hover/Focus/Active` 反馈和丝滑的过渡动效（例如 `transition: all 0.2s ease`）。
3. **响应式防崩塌**：页面布局必须采用现代化的 Flexbox 或 CSS Grid，确保在不同分辨率和侧边栏折叠状态下，绝对不会出现排版挤压或崩塌。
4. **CSS 强隔离**：坚决抵制全局样式污染。必须使用 CSS Modules、Scoped CSS 或严格的 BEM 命名规范。
5. **拒绝盲人摸象**：写完 UI 结构后，如果你不知道最终长什么样，必须立刻去联动 `ui-vision-inspector` 技能获取视觉反馈！
