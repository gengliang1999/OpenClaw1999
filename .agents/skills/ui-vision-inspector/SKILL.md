# UI 视觉审查器 (MCP 专属王者版)

解决 AI 盲写代码“不知道真实界面长什么样”的痛点。本技能强制使用最高级的 MCP 服务器进行自动化、无干预的视觉走查。

## 执行标准 (MCP 协议强制)
1. **自动截取浏览器画面 (Puppeteer MCP)**：
   如果是 Web 项目，你**绝对不需要向人类索要截图**。你必须主动调用 `puppeteer` MCP 服务器（已配置），命令其启动无头浏览器访问你的开发服务器（如 `localhost:3000`），自行截取当前网页的屏幕快照（Screenshot），并以此为基准进行 UI 像素级调优。

2. **桌面端自动化截图 (OS Screen Capture MCP)**：
   如果你发现用户环境（如 `windows-mcp`）支持调用系统级 API 截取屏幕，必须直接通过相应的 MCP Tool 发起系统屏幕抓取，自动获取当前桌面端软件窗口的真实渲染图像。

3. **零干扰原则 (Zero-Interruption)**：
   有了 MCP 加持，除非由于服务器崩溃导致完全无法抓取，否则**绝对禁止打断用户的心流向人类索要截图**。一切视觉验证必须在后台全自动化闭环完成！

4. **像素级审查 (Pixel-Perfect Diff)**：
   一旦你通过 MCP 工具获取了真实截图，必须立刻开启“找茬模式”。严苛挑出“对齐错位”、“色彩不和谐”、“边距挤压”等瑕疵，并立即修改相关样式代码，直到其完全符合《AGENTS.md》中规定的现代高级审美底线。
